## Code Examples

Here's a simple Go function that greets the world:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

And here's the same thing in Python:

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

if __name__ == "__main__":
    print(greet("World"))
```
